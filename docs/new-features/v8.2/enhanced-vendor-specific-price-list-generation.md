**Goal:** Functional

**Developer:**  Martin Schönbeck

**Feature Ticket:** [IDEMPIERE-4452](https://idempiere.atlassian.net/browse/IDEMPIERE-4452)

**Sponsor:** [Martin Schönbeck Beratungen GmbH](https://www.schoenbeck.de/)

**Description:**

To better support the generation of vendor specific price lists it was necessary, to allow prices to be included where the vendor is not the current vendor. Also the price calculation is often vendor specific and dependent on categorization done by the vendor. The 'Ignore current vendor flag' is shown only if vendor is selected.
