**Goal:** Functional

**Developer:** Nicolas Micoud, Carlos Ruiz

**Description:**

Now you can define a tree to be driven by its Search key

![01 TreeDriven](pathname:///img/new-features/v2.1/01_TreeDriven.png)

When you define the tree to be driven by search key, then new records will be *automagically* ordered in the tree according to its Search Key:

![02 TreeDriven](pathname:///img/new-features/v2.1/02_TreeDriven.png)

**NOTE:** If your Chart of Accounts is organized by the search key (account) then it's possible now to import directly the CSV from the Account Element window and the tree will be self-organized when defined this way.
