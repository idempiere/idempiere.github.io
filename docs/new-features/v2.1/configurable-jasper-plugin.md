**Goal:** Development

**Developer:** Thomas Bayen

**Description:**

The goal of this new feature is to allow the developer to create a custom replacement for the jasper plugin, this can be useful if you want to implement or test features not present in the core plugin, and/or if you need an older or newer version of jasper than the supported by the core plugin.
