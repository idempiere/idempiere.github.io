**Goal:** User Experience

**Description:**

In order to ease data entry without using the mouse, some new shortcuts were implemented in this version:

***Editing Window:***
- ***Shift+Alt+N:*** shortcut for new detail button

***Info Window:***
- ***Shift+Ctrl+Down:*** navigate to the results
- ***Enter:*** to select the result
- ***Escape:*** to cancel

You can see the full list of shortcuts and swipe events implemented at this [link](https://wiki.idempiere.org/en/Shortcuts)
