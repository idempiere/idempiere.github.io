**Goal:** Technical

**Developer:** Carlos Ruiz

**Description:**

Extending the functionality implemented for [Column Toolbar Button](https://wiki.idempiere.org/en/NF1.0_ColumnToolbarButton) you can now define those toolbar buttons to appear both, on toolbar and on the window at the same time.

This is defined at column level:

![01 BothButton](pathname:///img/new-features/v2.1/01_BothButton.png)

As a result on the example you're able to complete a Sales Order from the window, and also from the toolbar button (useful in case the button is not visible because the window is too large):

![02 BothButton](pathname:///img/new-features/v2.1/02_BothButton.png)
