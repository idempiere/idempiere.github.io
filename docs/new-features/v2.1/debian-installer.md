**Goal:** Technical

**Description:**

The new [IDempiere Debian Installer](https://wiki.idempiere.org/en/IDempiere_Debian_Installer) allows an almost automatic installation in Debian and Ubuntu systems.

The goal of this feature is to ask the less necessary to the user and make the proper validations to ensure a correct functioning of the system.

Please check the corresponding documentation page for prerequisites and values asked.

**Technical Info:** [IDEMPIERE-1900](http://idempiere.atlassian.net/browse/IDEMPIERE-1900)
