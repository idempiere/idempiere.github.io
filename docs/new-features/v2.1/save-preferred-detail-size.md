**Goal:** Usability

**Developer:** Nicolas Micoud, Carlos Ruiz

**Description:**

Now iDempiere is able to save the prefered detail size per user per window/tab.

Users don't need to do anything, is automatically saved when closing the window:

![01 DetailSize](pathname:///img/new-features/v2.1/01_DetailSize.png)

This information is saved as a preference:

![02 DetailSize](pathname:///img/new-features/v2.1/02_DetailSize.png)
