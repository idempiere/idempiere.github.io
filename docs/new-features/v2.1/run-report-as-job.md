**Goal:** User Experience

**Sponsor:** [TrekGlobal](http://www.trekglobal.com/)

**Description:**

Now the users can run reports and processes in background, just select the "Run as Job" flag and define your preferred notification method.

![01 RunReportAsJob](pathname:///img/new-features/v2.1/01_RunReportAsJob.png)

On the dashboard you can see now a "Running Jobs" and if you chose the notice method as notification your report will be attached to a notice in the corresponding button.

![02 RunReportAsJob](pathname:///img/new-features/v2.1/02_RunReportAsJob.png)
