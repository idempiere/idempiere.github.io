**Goal:** User Experience

**Sponsor:** [TrekGlobal](http://www.trekglobal.com/)

**Description:**

Now the users can add/remove the dashboard elements (gadgets) they want to see on the Home Screen.

Pushing the link *Manage gadgets* on the preference:

![NF20UserDashboard01](pathname:///img/new-features/v2.0/NF20UserDashboard01.png)

Will bring a form to allow the user to select the items to visualize/hide.

![NF20UserDashboard02](pathname:///img/new-features/v2.0/NF20UserDashboard02.png)

When you add a new gadget in *Dashboard Content* window, you can define if the gadget is automatically added to all users (depending on role configuration) with the flag *Show on Login*

![NF20UserDashboard03](pathname:///img/new-features/v2.0/NF20UserDashboard03.png)
