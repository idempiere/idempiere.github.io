**Goal:** User Experience

**Description:**

Now you can open several browser tabs and have different windows open on each tab.

NOTE that all tabs must be on the same session (user+role), multiple sessions within the same browser are not supported, for that you must use multiple browsers, or different profile (like incognito mode).
