**Goal:** Usability

**Developers:** Nicolas Micoud

**Description:**

A new button to add comments to records, like a PostIt, has been added to the toolbar in every window.

![01 PostIt](pathname:///img/new-features/v5.1/01_PostIt.png)

The difference with the Record Comments button (Chat) is that the PostIt is a single record and is editable.
