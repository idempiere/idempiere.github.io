**Goal:** Usability

**Developer:** Le Quy Hiep

**Description:**

For a better user experience, some improvements were made to tab behavior:
- it is possible now to drag and drop tabs to reorder them
- when opening a new tab (for example via zoom feature) the tab is opened at right of the current tab
- when closing a tab, the previous active tab is focused
