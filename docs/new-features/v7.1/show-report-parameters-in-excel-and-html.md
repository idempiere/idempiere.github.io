**Goal:** User Experience

**Developer:** Heng Sin Low

**Description:**

iDempiere can show now the execution parameters when reports are exported to excel and HTML.

![ShowReportParams](pathname:///img/new-features/v7.1/ShowReportParams.png)

This is the result in HTML:

![ShowReportParamsHTML](pathname:///img/new-features/v7.1/ShowReportParamsHTML.png)

And this is the result in excel as shown with libreoffice:

![ShowReportParamExcel](pathname:///img/new-features/v7.1/ShowReportParamExcel.png)

Please note that PDF has always shown the parameters, and CSV is not showing parameters as CSV format is used many times for information exchange with other systems.
