**Goal:** User Experience

**Sponsor:** [TrekGlobal](http://www.trekglobal.com/)

**Description:**

When adding a new record on a table, the related lists don't need to be refreshed, the new record is automagically added to the list.

For example, when you are on Sales Order Line, zoom to Charge and create a new one, when you return to Sales Order window, the charge will be available on the list without the need to call the refresh link.
