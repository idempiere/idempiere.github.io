**Goal:** Usability

**Contributor:** Quality Systems & Solutions (GlobalQSS)

**Description:**

This feature shows the user a history of recent records that been have created or updated.

![Screenshot9](pathname:///img/new-features/v1.0/Screenshot9.png)

It is possible to delete a item of the list (drag item to trash).  The list of items is updated automatically.

![Screenshot10](pathname:///img/new-features/v1.0/Screenshot10.png)

Additionally, a record can be edited without saving changes and it will be automatically added to the stack list.
