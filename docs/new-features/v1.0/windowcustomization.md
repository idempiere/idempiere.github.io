**Goal:** Usability

**Contributor:** Dirk Niemeyer

**Description:**

Customize for user the columns to show and the order in the window on grid mode

This button is just active in grid mode

![custom1](pathname:///img/new-features/v1.0/custom1.png)

Click in the button appear a popup with the columns that can choose what columns to show in what order want to appear the columns.
