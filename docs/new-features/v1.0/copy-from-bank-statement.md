**Goal:** Functional

**Contributor:** Nicolas Micoud

**Description:**

This new feature adds CopyFrom on Bank Statement.

Should only copy :

- Lines without payment.

- Line with a payment (only if this payment is not on another CO/CL/DR bank statement).
