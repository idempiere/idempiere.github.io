**Goal:** Usability

**Sponsor:** [Trek Global](https://www.trekglobal.com)

**Description:**

- The menu tree has been moved to the top panel, in order to make it accessible even if the left panel is collapsed, and now the menu is shown as a popup window when the user press the Menu button on top.

- Note also the favourites and menu links related to windows have now a little icon in front, pushing this icon open the window in *Adding New Record* mode.

- There is also a button added to the menu window that allows user to filter the menu options to show or hide reports, processes, workflows and windows/forms.  This button also allows the user to choose a flat view of the menu (instead of tree view)
