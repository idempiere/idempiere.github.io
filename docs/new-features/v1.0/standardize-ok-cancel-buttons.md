**Goal:** Usability

**Sponsor:** [Trek Global](https://www.trekglobal.com)

**Description:**

For usability is important to always see the OK and Cancel buttons in the same order.

Before this change, buttons would sometimes appear organized in certain position, and sometimes in another. We've tried to establish a standard in doing this in:

- Process Dialog
- Find Window
- Message box
