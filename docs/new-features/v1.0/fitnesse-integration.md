**Goal:** Functional Testing

**Sponsor:** [Trek Global](https://www.trekglobal.com)

**Description:**

Allow user and developer to create and execute Fitnesse test within the iDempiere environment.

## Running From Eclipse
- Import the new "fitnesse" java project into your Eclipse workspace. This is the Fitnesse wiki server that you can run from Eclipse or command line.  ([Click here if you are confused](https://wiki.idempiere.org/en/Talk:NF001_Fitnesse_Integration#My_mistake_looking_for_Fitnesse_java_project).)

- Start the iDempiere server

- Start the Fitnesse bundles to accept request from the Fitnesse wiki server

![List bundles](pathname:///img/new-features/v1.0/List-bundles.png)

![Fitnesse bundles id](pathname:///img/new-features/v1.0/Fitnesse-bundles-id.png)

![Start fitnesse bundles](pathname:///img/new-features/v1.0/Start-fitnesse-bundles.png)


- Start the Fitnesse wiki server

![Run fitnesse](pathname:///img/new-features/v1.0/Run-fitnesse.png)

![Fitnesse started](pathname:///img/new-features/v1.0/Fitnesse-started.png)


- Edit global setting

![Global setting](pathname:///img/new-features/v1.0/Global-setting.png)

- Run Fitnesse test

![Quicktest result](pathname:///img/new-features/v1.0/Quicktest-result.png)
