**Goal:** Functional

**Sponsor:** [TrekGlobal](http://www.trekglobal.com/)

**Description:**

Closing and Voiding documents can be a dangerous not-reversible operation.

Now you can configure per client if an additional confirmation must be asked to the user closing or voiding a document.

This new option can be configured at [Client Info](https://wiki.idempiere.org/en/Client_(Window_ID-109)#TAB:_Client_Info) tab:

![NF10ConfirmDocVoidClose01](pathname:///img/new-features/v1.0/NF10ConfirmDocVoidClose01.png)

And then when a user tries to close or void it's asked to confirm:

![NF10ConfirmDocVoidClose02](pathname:///img/new-features/v1.0/NF10ConfirmDocVoidClose02.png)

![NF10ConfirmDocVoidClose03](pathname:///img/new-features/v1.0/NF10ConfirmDocVoidClose03.png)
