**Goal:** Functional

**Sponsor:** [Trek Global](https://www.trekglobal.com)

**Description:**

The "Request Calendar" allows the user to create, view and query quickly any request.

![screenshot1](pathname:///img/new-features/v1.0/screenshot1.png)

Add new request.

Click on the day, in which you want to create the new request

![screenshot2](pathname:///img/new-features/v1.0/screenshot2.png)

 * Define the values ​​for the request

![screenshot3](pathname:///img/new-features/v1.0/screenshot3.png)

 * The new request, shown in calendar

![screenshot4](pathname:///img/new-features/v1.0/screenshot4.png)

It is possible to see the summary of the request by clicking on items in the calendar. For more detailed information you can click on the "Zoom" button , this will display the request in "Request All" window
