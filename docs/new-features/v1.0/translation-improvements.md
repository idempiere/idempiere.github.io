**Goal:** Technical

**Sponsor:** [TrekGlobal](http://www.trekglobal.com/)

**Description:**

Now you can import/export translation directly from the webui client, to import you can choose a server folder to import the translation, or upload a zipped file with the translation files.

Samely for export you can choose a server folder, or a zip file and download directly to your computer.
