**Goal:** Functional

**Developer:** Carlos Ruiz

**Description:**

The documents Production and Bank Statement can be configured now to use different bank statements, this provides better flexibility to configure the template for DocumentNo based on different sequences, and also allows to set the DocumentNo and Date when the document is completed.
