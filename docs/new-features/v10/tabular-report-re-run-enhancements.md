**Goal:** UX/Functional

**Developer:** cloudempiere

**Feature Ticket:** [IDEMPIERE-5275](https://idempiere.atlassian.net/browse/IDEMPIERE-5275)

**Description:**

Until now, tabular reports are opened in new tab - as result opening many tabs. This enhancement allow open Tabular Reports in same Tab by replacing content, instead opening report result in new one.

**Changes:**

- New icon **Re-Run** in Report toolbar - popup Report

- New sysconfig: ZK_REPORT_TABLE_OPEN_IN_NEW_TAB - allow setup legacy behavior
