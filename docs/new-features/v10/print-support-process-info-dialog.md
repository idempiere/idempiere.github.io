**Developer:** cloudempiere, Takacs Peter, Review: CarlosRuiz, HengsinLow

**Feature Ticket: [IDEMPIERE-5356](https://idempiere.atlassian.net/browse/IDEMPIERE-5356)**

**Changes**

- New button added to Process Info Dialog, allowing print documents after generate some documents (eg. from info window)

- Deprecated generate shipment, invoice (manual) deprecate also automatic print functionality. This button is a replacement in this use case.

**Notes**

- when multiple documents are generated, the print button prints all documents/pages in the same pdf

- the print is not automatic, but user choice
