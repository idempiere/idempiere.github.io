**Feature Ticket:** [IDEMPIERE-5265](https://idempiere.atlassian.net/browse/IDEMPIERE-5265)

**Description:**
Selecting items from a multiple selection list field can be time consuming, especially if you want to order it properly.
To resolve this, an editor has been added ; it can be opened using a right click on the field and selecting "Assistant" (available from fields on windows and process parameters).

![NF10 WizardMultipleListPopup](pathname:///img/new-features/v10/NF10_WizardMultipleListPopup.png)

The panel shows all available items. And you can drag-and-drop (or double-click) to select those to be set in the field
