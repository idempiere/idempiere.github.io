**Goal:** Functional

**Developer:** Diego Ruiz

**Sponsor:** [BX Service GmbH](https://www.bx-service.com/)

**Description:**

This new feature introduces the UOM and Qty Entered fields in Inventory Move Line. Previously, quantities could only be entered using the product's default UOM. With this update, you can now move inventory using different UOMs, and the system will automatically calculate the movement quantity.

This enhancement aligns the functionality of Inventory Move with documents like Material Receipt, offering greater flexibility and consistency.
