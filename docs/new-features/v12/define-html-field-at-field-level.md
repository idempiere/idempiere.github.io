**Description:**

https://idempiere.atlassian.net/browse/IDEMPIERE-6147 adds a HTML listbox (visible when Reference Overwrite is filled) on
- Window > Tab > Field
- Window Customization > Tab > Field

![HTML Field](pathname:///img/new-features/v12/HTML_Field.png)

So now, you can have a column which can be displayed as plain text or HTML according to the setup done on the field !
