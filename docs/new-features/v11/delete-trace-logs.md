The purpose of this improvement is to delete old trace logs periodically that generated daily.

the process can triggered manually using menu "Delete Trace Logs" with parameter "Keep Log Days". it also can run periodically using Scheduler and set "is active" as checked. as default, the scheduler is inactive.
