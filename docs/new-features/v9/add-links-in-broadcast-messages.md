**Goal:** Usability

**Developer:**  Nicolas Micoud - original credits go to Matheus E. Marcelino and Murilo H. Torquato

**Feature Ticket:** [IDEMPIERE-5135](https://idempiere.atlassian.net/browse/IDEMPIERE-5135)

**Description:**

Is now possible to add link in broadcast messages to open a record within you current session.

When the message is displayed the link appears as a standard HTML link:

![BroadcastMessageLink Panel](pathname:///img/new-features/v9/BroadcastMessageLink_Panel.png)

When you click on the link, it opens the window attached to the record:

![BroadcastMessageLink Window](pathname:///img/new-features/v9/BroadcastMessageLink_Window.png)

Adding link is not easy for end users, is better to add them programmaticaly, you can find an example in the [ticket](https://idempiere.atlassian.net/browse/IDEMPIERE-5135).
