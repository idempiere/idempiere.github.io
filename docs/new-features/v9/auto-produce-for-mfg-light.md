**Goal:** Usability

**Developer:** Hengsin ([talk](https://wiki.idempiere.org/en/User_talk:Hengsin))

**Feature Ticket:** [IDEMPIERE-5031](https://idempiere.atlassian.net/browse/IDEMPIERE-5031)

**Description:**

- Add Auto Produce flag to Product window (for BOM product)
- Add read only Shipment Line reference to Production window
- Add read only Auto Produce flag to Shipment Line tab at Shipment (Customer) window
- When auto produce flag is check, the system will auto create and complete a Production document after user completes a Shipment document for the product (with a validated BOM).
