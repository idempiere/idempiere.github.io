**Goal:** Functional

**Developer:**  Logilite Technologies

**Feature Ticket:** [IDEMPIERE-4882](http://idempiere.atlassian.net/browse/IDEMPIERE-4882)

**Sponsors:** [Logilite Technologies LLP](http://www.logilite.com)

**Description:**

Before this enhancement, Inventory move has no restriction on warehouse. As part of this ticket, Warehouse and Warehouse To field are added on inventory move header. This field are primarily used for filtering Locator on Locator and Locator To field on inventory move line.

![Warehouse Inventory Move](pathname:///img/new-features/v9/Warehouse-Inventory-Move.png)

Warehouse on header are not mandatory. if not selected, it let you use locator from all warehouse.
