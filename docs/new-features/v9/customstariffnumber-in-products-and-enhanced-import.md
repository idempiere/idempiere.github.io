**Sponsor:** [Martin Schönbeck Beratungen GmbH](https://www.schoenbeck.de/)

**Description:**

Because it's often necessary for statistical and customs reasons to know the customs tariff number (often also known harmonized system number) or similar numbers this column is added to m_product. To be able to import it easily it is also added to i_product. And being at this also Group1 and Group2 are added to i_product so that they can be imported easily, too.
