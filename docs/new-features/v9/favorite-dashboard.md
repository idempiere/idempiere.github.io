**Developer:**  Logilite Technologies

**Feature Ticket:** [IDEMPIERE-3340](http://idempiere.atlassian.net/browse/IDEMPIERE-3340)

**Sponsors:** [Logilite Technologies LLP](http://www.logilite.com), [Adaxa](http://www.adaxa.com/)

**Description:**

Logilite has developed user organizable favorite tree dashboard widget in Adaxa suite.  The feature is ported to iDempiere by Logilite Technologies and improved to integrate it to default favorite panel, Migrating existing favorite data stored in old format to new tree structural, supporting auto launch configuration from favorite panel itself, making mobile friendly.

Following are feature of user favorite panel
1. Drag and drop to organize structure
1. Tree structure support
1. Allow to manage auto launch sequence

Below screen show shows enhanced user favorite panel. This enhancement available from iDempiere 9 version.
