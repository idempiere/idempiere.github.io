New Accounting Dimensions

Feature: New Accounting Dimensions

Goal: Functional

Developer: Deepak Pansheriya

Feature Ticket: IDEMPIERE-5598

Description​

in iDempiere, Activity use for cost center as well as department. But this needs to customize. So it is better to add out of box dimensions for Department and cost center.

As part of this feature, added following accounting dimensions

  1. Charge: There may be cases when same GL account can be linked to multiple charges. In this case charge as dimnsion required to summaries accounting per charge.
  2. Asset: Adding asset as accounting dimension allow to know expense, depreciations, asset value without adding new GL accounting.
  3. Department: Activity is used till date and it requires customization to rename activity system element. In this feature added new window for Department and Department as accounting dimension as out of box.
  4. Warehouse: Adding warehouse as out of box accounting dimension allow to trek inventory value per warehouse using financial report.
  5. Employee: For trekking Employee salary expense or observed overhead difference analysis.
  6. Cost Center: Implementation was using Activity, organization or custom column as cost center. but this all approach required customization. after adding out of box cost center dimension as well as window to maintain cost center reduce to customization efforts.
  7. Attribute Set Instance: When batch wise inventory is important to manage, adding ASI as accounting dimension allow to easily trek inventory valuation report and inventory accounting difference.
  8. Custom Field: This allows to configure text type field as accounting dimension.
  9. Tax: This feature has added tax as parameter on accounting related reports so that tax related statements can be produced.
danger

Following new windows are added - action required for existing implementations

  • Department: A dedicated Department window is now available. If you previously renamed the Activity window or system element to "Department", you must rename it back to "Activity".
  • Cost Center: A dedicated Cost Center window is now available. If you previously renamed Activity to "Cost Center", you must rename it back to "Activity".

Following new Context Variable added to track dimension used for accounting

  • $Element_AI - For Attribute Set Instance
  • $Element_AS - For Asset
  • $Element_CC - For Cost Center
  • $Element_CH - For Charge
  • $Element_DP - For Department
  • $Element_EP - For Employee
  • $Element_F1 - For Custom Field 1
  • $Element_F2 - For Custom Field 2
  • $Element_F3 - For Custom Field 3
  • $Element_F4 - For Custom Field 4
  • $Element_WH - For Warehouse
