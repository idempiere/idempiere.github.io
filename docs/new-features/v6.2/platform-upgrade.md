**Goal:** Architecture

**Developers:** Le Quy Hiep,  Heng Sin Low,  Carlos Ruiz

**Description:**

Release 6.2 included important upgrades:

***Platform***
- The project now run on Eclipse SimRel 201809

***ZK***
- [Zk8.6.0.1](https://www.zkoss.org/product/zk/releasenote/8.6.0.1)

***Java***
- Project now must be compiled with [Java 11](https://openjdk.java.net/projects/jdk/11/)

***Web Server/Servlet Container***
- Jetty 9.4.7

***Assembly***
- Maven
Assembly of the project was moved from buckminster to maven

***Libraries***
- Many libraries updated
