**Goal:** Usability

**Developers:** Deepak Pansheriya, Carlos Ruiz

**Description:**

The role panel after the user provides the login credentials has been modified.

Now the Client, Role and Organization fields become read-only if there is just one option for the user to select.

Also, the Date field can become read-only if the SysConfig key [System_Configurator_(Window_ID-50006)#ALogin_ShowDate|ALogin_ShowDate] is set to N.
