# User

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 108</span>

*07/06/1999 → 03/06/2021*

![User](./img/user.png)

**Description:** Maintain Users of the system

**Comment/Help:** The User Window allows you to maintain User of the system.  Users can log into the system and have access to functionality via one or more roles.  A user can also be a business partner contact.
