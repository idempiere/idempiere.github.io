## Tab: Web Store

*Tab Level 0 · Created 02/05/2005 · Updated 10/03/2022*

**Description:** Define Web Store

**Comment/Help:** Define the web store settings. The web server context must be unique and determines the settings used in the actual transaction. The selected warehouse determines the organization.  The email settings are tested with the tenant email settings.
