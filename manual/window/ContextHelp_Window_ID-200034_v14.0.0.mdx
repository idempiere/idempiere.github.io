# Context Help

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 200034</span>

*05/02/2013 → 08/02/2013*

![Context Help](/img/manual/ContextHelp-Window_iDempiere_v14.0.0.png)

**Description:** Maintain Context Helps

**Comment/Help:** The Context Help window defines the context help message text to display in the help pane. It is for Web UI only.

## Tab: Context Help

*Tab Level 0 · Created 05/02/2013 · Updated 08/02/2013*

**Description:** Context Helps

**Comment/Help:** The Context Help tab defines the type of context help

| **Name** | **Description** | **Comment/Help** | **Technical Data** |
|---|---|---|---|
| Tenant | Tenant for this installation. | A Tenant is a company or a legal entity. You cannot share data between Tenants. | AD_CtxHelp.AD_Client_ID<small> numeric(10) <br/> Table Direct</small> |
| Organization | Organizational entity within tenant | An organization is a unit of your tenant or legal entity - examples are store, department. You can share data between organizations. | AD_CtxHelp.AD_Org_ID<small> numeric(10) <br/> Table Direct</small> |
| Name | Alphanumeric identifier of the entity | The name of an entity (record) is used as an default search option in addition to the search key. The name is up to 60 characters in length. | AD_CtxHelp.Name<small> character varying(120) <br/> String</small> |
| Description | Optional short description of the record | A description is limited to 255 characters. | AD_CtxHelp.Description<small> character varying(255) <br/> Text</small> |
| Active | The record is active in the system | There are two methods of making records unavailable in the system: One is to delete the record, the other is to de-activate the record. A de-activated record is not available for selection, but available for reports.<br/>There are two reasons for de-activating and not deleting records:<br/>(1) The system requires the record for audit purposes.<br/>(2) The record is referenced by other records. E.g., you cannot delete a Business Partner, if there are invoices for this partner record existing. You de-activate the Business Partner and prevent that this record is used for future entries. | AD_CtxHelp.IsActive<small> character(1) <br/> Yes-No</small> |
| Context Type | Type of Context Help | The Context Type field indicates the type of context help (tab, process, form etc) this context help is defined as | AD_CtxHelp.CtxType<small> character(1) <br/> List</small> |
