# Inventory Move

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 170</span>

*19/12/1999 → 03/06/2021*

![Inventory Move](/img/manual/InventoryMove-Window_iDempiere_v14.0.0.png)

**Description:** Inventory Move

**Comment/Help:** The Inventory Move Window allows you to move inventory from one location or warehouse to another.  After the appropriate lines are entered selecting the 'Process Movements' button will adjust the inventory.
