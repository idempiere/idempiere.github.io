# Import Report Line Set

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 249</span>

*11/01/2003 → 02/01/2000*

![Import Report Line Set](/img/manual/ImportReportLineSet-Window_iDempiere_v14.0.0.png)

**Description:** Import Report Line Sets

**Comment/Help:** The Import Report Line Set Window is an interim table which is used when importing external data into the system.  Selecting the 'Process' button will either add or modify the appropriate records.
