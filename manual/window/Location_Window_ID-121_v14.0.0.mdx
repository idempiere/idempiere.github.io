# Location

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 121</span>

*11/06/1999 → 02/01/2000*

![Location](/img/manual/Location-Window_iDempiere_v14.0.0.png)

**Description:** Maintain Location Address

**Comment/Help:** The Location Window defines the address data within the system.  This window is for System Admin only.  Users would access location entries using the location button or tab on the appropriate window (i.e Order Entry or Business Partner)

## Tab: Location

*Tab Level 0 · Created 23/06/1999 · Updated 02/01/2000*
