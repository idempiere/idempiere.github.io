# Country Region and City

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 122</span>

*11/06/1999 → 02/01/2000*

![Country Region and City](/img/manual/CountryRegionandCity-Window_iDempiere_v14.0.0.png)

**Description:** Maintain Countries Regions and Cities

**Comment/Help:** The Countries, Regions and Cities Window defines the different entities that can be used in any address field.  It defines the format of the address as well as associating Regions with Countries and Cities with Regions or Countries. 
&lt;p&gt;You would define Countries usually only on System level.
