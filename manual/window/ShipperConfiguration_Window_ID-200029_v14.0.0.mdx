## Tab: Shipper

*Tab Level 0 · Created 13/12/2012 · Updated 13/12/2012*

**Description:** Shippers

**Comment/Help:** The Shippers Tab defines any entity who will provide shipping to or shipping from an Organization.

| **Name** | **Description** | **Comment/Help** | **Technical Data** |
|---|---|---|---|
| Tenant | Tenant for this installation. | A Tenant is a company or a legal entity. You cannot share data between Tenants. | M_ShipperCfg.AD_Client_ID<small> numeric(10) <br/> Table Direct</small> |
| Organization | Organizational entity within tenant | An organization is a unit of your tenant or legal entity - examples are store, department. You can share data between organizations. | M_ShipperCfg.AD_Org_ID<small> numeric(10) <br/> Table Direct</small> |
| Name | Alphanumeric identifier of the entity | The name of an entity (record) is used as an default search option in addition to the search key. The name is up to 60 characters in length. | M_ShipperCfg.Name<small> character varying(60) <br/> String</small> |
| Description | Optional short description of the record | A description is limited to 255 characters. | M_ShipperCfg.Description<small> character varying(255) <br/> String</small> |
| Active | The record is active in the system | There are two methods of making records unavailable in the system: One is to delete the record, the other is to de-activate the record. A de-activated record is not available for selection, but available for reports.<br/>There are two reasons for de-activating and not deleting records:<br/>(1) The system requires the record for audit purposes.<br/>(2) The record is referenced by other records. E.g., you cannot delete a Business Partner, if there are invoices for this partner record existing. You de-activate the Business Partner and prevent that this record is used for future entries. | M_ShipperCfg.IsActive<small> character(1) <br/> Yes-No</small> |
| Online Access | Can be accessed online  | The Online Access check box indicates if the application can be accessed via the web.  | M_ShipperCfg.IsOnline<small> character(1) <br/> Yes-No</small> |

| Service Code |  |  | M_ShipperCfg.ShippingServiceCode<small> character varying(40) <br/> String</small> |
| Tracking URL | URL of the shipper to track shipments | The variable @TrackingNo@ in the URL is replaced by the actual tracking number of the shipment. | M_ShipperCfg.TrackingURL<small> character varying(120) <br/> URL</small> |
| International |  |  | M_ShipperCfg.IsInternational<small> character(1) <br/> Yes-No</small> |
| Saturday Delivery |  |  | M_ShipperCfg.IsSaturdayDelivery<small> character(1) <br/> Yes-No</small> |
