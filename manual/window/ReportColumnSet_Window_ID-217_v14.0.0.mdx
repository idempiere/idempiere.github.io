# Report Column Set

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 217</span>

*13/05/2001 → 02/01/2000*

![Report Column Set](/img/manual/ReportColumnSet-Window_iDempiere_v14.0.0.png)

**Description:** Maintain Financial Report Column Sets

**Comment/Help:** The Report Column Set defines what data is printed in the columns of a report.

## Tab: Report Column Set

*Tab Level 0 · Created 13/05/2001 · Updated 02/01/2000*

**Description:** Maintain Financial Report Column Sets

**Comment/Help:** Column Sets are the combination of Columns to be included in a Financial Report

| **Name** | **Description** | **Comment/Help** | **Technical Data** |
|---|---|---|---|
| Tenant | Tenant for this installation. | A Tenant is a company or a legal entity. You cannot share data between Tenants. | PA_ReportColumnSet.AD_Client_ID<small> numeric(10) <br/> Table Direct</small> |
| Organization | Organizational entity within tenant | An organization is a unit of your tenant or legal entity - examples are store, department. You can share data between organizations. | PA_ReportColumnSet.AD_Org_ID<small> numeric(10) <br/> Table Direct</small> |
| Name | Alphanumeric identifier of the entity | The name of an entity (record) is used as an default search option in addition to the search key. The name is up to 60 characters in length. | PA_ReportColumnSet.Name<small> character varying(60) <br/> String</small> |
| Description | Optional short description of the record | A description is limited to 255 characters. | PA_ReportColumnSet.Description<small> character varying(255) <br/> String</small> |
| Active | The record is active in the system | There are two methods of making records unavailable in the system: One is to delete the record, the other is to de-activate the record. A de-activated record is not available for selection, but available for reports.<br/>There are two reasons for de-activating and not deleting records:<br/>(1) The system requires the record for audit purposes.<br/>(2) The record is referenced by other records. E.g., you cannot delete a Business Partner, if there are invoices for this partner record existing. You de-activate the Business Partner and prevent that this record is used for future entries. | PA_ReportColumnSet.IsActive<small> character(1) <br/> Yes-No</small> |
| Copy Columns | Copy Report Columns from other Column Set | Copy columns at the end of this Column Set. Please note that you need to re-set the calculation operands. | PA_ReportColumnSet.Processing<small> character(1) <br/> Button</small> |
