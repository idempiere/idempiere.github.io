# Report and Process

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 165</span>

*11/12/1999 → 08/12/2023*

![Report and Process](/img/manual/ReportandProcess-Window_iDempiere_v14.0.0.png)

**Description:** Maintain Reports &amp; Processes

**Comment/Help:** The Report &amp; Process Window is used to define the parameters and access rules for every Report and Process within the system.  This window is for System Admin use only.

## Tab: Report &amp; Process

*Tab Level 0 · Created 11/12/1999 · Updated 02/01/2000*

**Description:** Define Report &amp; Process

**Comment/Help:** The Report and Process Tab defines each report and process run in the system.  
Please note that the Print Format is only used for Document Print when you want to share a common format defined on System level. For normal reports and formats, you would not define a Print Format here, but modify the generated print format and set it as the default.
