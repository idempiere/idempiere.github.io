## Tab: › Format Item

*Tab Level 1 · Created 11/07/2002 · Updated 15/01/2024*

**Description:** Print Format Item

**Comment/Help:** Item in the print format maintaining layout information.  The following additional variables are available when printing:
&lt;p&gt;@*Page@ - the current page number
&lt;br&gt;@*PageCount@ - the total number of pages
&lt;br&gt;@*MultiPageInfo@ - "Page x of y" - only printed if more than one page
&lt;br&gt;@*CopyInfo@ - if it is a copy of the document "Duplicate" is printed
&lt;br&gt;@*ReportName@ - name of the report
&lt;br&gt;@*Header@ - Full Header with user/tenant/org name and connection
&lt;br&gt;@*CurrentDate@ - the print date
&lt;br&gt;@*CurrentDateTime@ - the print time

&lt;p&gt;The layout logic for forms is: &lt;br&gt;
- if not printed: ignore &lt;br&gt;
- set New Line Position &lt;br&gt;
- new page &lt;br&gt;
- new line, if something was printed since last new line&lt;br&gt;
- format item&lt;br&gt;
- print it (if not null/empty selected)&lt;br&gt;
(i.e. suppress null does not prevent new line)
