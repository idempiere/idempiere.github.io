<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 172</span>

**Description:** Import Business Partner

**Comment/Help:** The Import Business Partner Window is an interim table which is used when importing external data into the system.  Selecting the 'Process' button will either add or modify the appropriate records.
