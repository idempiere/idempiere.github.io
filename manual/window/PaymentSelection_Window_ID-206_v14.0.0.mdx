# Payment Selection

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 206</span>

*15/02/2001 → 03/06/2021*

![Payment Selection](/img/manual/PaymentSelection-Window_iDempiere_v14.0.0.png)

**Description:** Select Invoices for Payment

**Comment/Help:** The Payment Selection Window allows you to select and process the invoices you want to pay.
(a) You can create a Payment Selection and select the invoices manually or generate the suggestions based on criteria.  You can change the amount to be paid or delete invoices, you do not want to pay.

(b) If you are happy with the Payment Selection, you create the payments.

(c) You print or export your payments in the Window Payment Print/Export
