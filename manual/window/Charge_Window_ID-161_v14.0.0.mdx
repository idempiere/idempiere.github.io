# Charge

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 161</span>

*04/12/1999 → 02/01/2000*

![Charge](/img/manual/Charge-Window_iDempiere_v14.0.0.png)

**Description:** Maintain Charges

**Comment/Help:** The Charges Window defines the different charges that may be incurred.  These can include Bank Charges, Vendor Charges and Handling Charges.

## Tab: Charge

*Tab Level 0 · Created 04/12/1999 · Updated 19/01/2006*

**Description:** Charge

**Comment/Help:** The Charge Tab defines the unique charges that may be associated with a document.
The optional Business Partner allows to create Purchase Orders from Requisitions.
