## Tab: Request Processor

*Tab Level 0 · Created 11/01/2001 · Updated 02/01/2000*

**Description:** Request Processor

**Comment/Help:** The Request Processor Tab allows you to define processes that you want to occur and the frequency and timing of these processes. If no other user is found, the items are assigned to the supervisor. A Request Processor can be just for a specific Request Type or for all.
