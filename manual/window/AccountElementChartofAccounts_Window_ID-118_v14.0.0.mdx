# Account Element (Chart of Accounts)

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 118</span>

*11/06/1999 → 08/12/2023*

![Account Element (Chart of Accounts)](/img/manual/AccountElementChartofAccounts-Window_iDempiere_v14.0.0.png)

**Description:** Maintain Account Elements

**Comment/Help:** The Account Element Window is used to define and maintain the Accounting Element and User Defined Elements. 
One of the account segments is your natural account segment (Chart of Account). You may add a new account element for parallel reporting or for user defined accounting segments.
