## Tab: Accounting Processor

*Tab Level 0 · Created 19/02/2004 · Updated 02/01/2000*

**Description:** Maintain Accounting Processor/Server Parameters

**Comment/Help:** Accounting Processor/Server Parameters. If no account schema is selected, all accounting schema are processed. If no transaction table is selected, accounting for all transaction is created.

| **Name** | **Description** | **Comment/Help** | **Technical Data** |

| Description | Optional short description of the record | A description is limited to 255 characters. | C_AcctProcessor.Description<small> character varying(255) <br/> String</small> |
| Active | The record is active in the system | There are two methods of making records unavailable in the system: One is to delete the record, the other is to de-activate the record.
