# Payments into Batch

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 200031</span>

*23/01/2013 → 11/04/2013*

![Payments into Batch](/img/manual/PaymentsintoBatch-Window_iDempiere_v14.0.0.png)

**Description:** Process Payments

**Comment/Help:** The Process Payments Window allows you to enter payments for invoices.  If the payment is for a single invoice then it can be processed here.
