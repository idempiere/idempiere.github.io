**Comment/Help:** The Tax Category Window is used to enter and maintain Tax Categories.  Each product is associated with a tax category which facilitates reacting changing tax rates.

## Tab: Tax Category

*Tab Level 0 · Created 09/08/1999 · Updated 03/03/2008*

**Description:** Tax Category

**Comment/Help:** The Tax Category Tab is used to define and maintain Tax Categories.  Each Product is associated with a Tax Category.  This facilitates adapting to changes in taxation.

| **Name** | **Description** | **Comment/Help** | **Technical Data** |
|---|---|---|---|
| Tenant | Tenant for this installation. | A Tenant is a company or a legal entity. You cannot share data between Tenants. | C_TaxCategory.AD_Client_ID<small> numeric(10) <br/> Table Direct</small> |
| Organization | Organizational entity within tenant | An organization is a unit of your tenant or legal entity - examples are store, department. You can share data between organizations. | C_TaxCategory.AD_Org_ID<small> numeric(10) <br/> Table Direct</small> |
| Name | Alphanumeric identifier of the entity | The name of an entity (record) is used as an default search option in addition to the search key. The name is up to 60 characters in length. | C_TaxCategory.Name<small> character varying(60) <br/> String</small> |
| Description | Optional short description of the record | A description is limited to 255 characters. | C_TaxCategory.Description<small> character varying(255) <br/> String</small> |
| Active | The record is active in the system | There are two methods of making records unavailable in the system: One is to delete the record, the other is to de-activate the record. A de-activated record is not available for selection, but available for reports.<br/>There are two reasons for de-activating and not deleting records:<br/>(1) The system requires the record for audit purposes.<br/>(2) The record is referenced by other records. E.g., you cannot delete a Business Partner, if there are invoices for this partner record existing. You de-activate the Business Partner and prevent that this record is used for future entries. | C_TaxCategory.IsActive<small> character(1) <br/> Yes-No</small> |
| Default | Default value | The Default Checkbox indicates if this record will be used as a default value. | C_TaxCategory.IsDefault<small> character(1) <br/> Yes-No</small> |
