## Tab: Attribute Set

*Tab Level 0 · Created 05/05/2003 · Updated 16/03/2009*

**Description:** Maintain Product Attribute Set

**Comment/Help:** Define Product Attribute Sets to add additional attributes and values to the product. You need to define an Attribute Set if you want to enable Serial and Lot Number and Guarantee Date tracking.  Note that the Guarantee Days here determine the Shelf Life of a product instance after manufacturing (the Guarantee Days on the product determines a Customer Service date after selling)
If the Attribute Set is mandatory, a product instance needs to be selected/created before shipping.
