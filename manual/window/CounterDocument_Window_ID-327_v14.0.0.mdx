# Counter Document

<span className="badge badge--primary">Window</span> <span className="badge badge--secondary">ID 327</span>

*14/04/2004 → 02/01/2000*

![Counter Document](/img/manual/CounterDocument-Window_iDempiere_v14.0.0.png)

**Description:** Maintain Counter Document Types

**Comment/Help:** When using explicit documents for inter-org transaction (after linking a Business Partner to an Organization), you can determine what document type the counter document is based on the document type of the original transaction.  Example: a "Standard Order" creates a "Standard PO". 
If you define a relationship here, you overwrite the default counter document type in the Document Type definition. This allows you to define a specific mapping.
If not defined, the first matching document type is used (e.g. AR Invoice - first AP Invoice document type)

**Description:** Maintain Counter Document Types

**Comment/Help:** When using explicit documents for inter-org transaction (after linking a Business Partner to an Organization), you can determine what document type the counter document is based on the document type of the original transaction.  Example: a "Standard Order" creates a "Standard PO". 
If you define a relationship here, you overwrite the default counter document type in the Document Type definition. This allows you to define a specific mapping.
You can define conter documents for all organizations (of the original transaction) or for a specific organization.
