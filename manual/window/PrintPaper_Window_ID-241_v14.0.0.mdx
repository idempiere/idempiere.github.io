## Tab: Print Paper

*Tab Level 0 · Created 11/07/2002 · Updated 18/01/2006*

**Description:** Maintain Print Paper

**Comment/Help:** Printer Paper Size, Orientation and Margins. 
The Validation Code contains the Media Size name. The names of the media sizes correspond to those in the IPP 1.1 RFC 2911 - http://www.ietf.org/rfc/rfc2911.txt  &lt;br&gt;
If the name is not found, the SizeX/Y with the dimension is used. Size and dimension are ignored, if a valid media size name is used.
