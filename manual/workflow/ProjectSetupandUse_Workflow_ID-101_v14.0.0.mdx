# Project Setup and Use

<span className="badge badge--primary">Workflow</span> <span className="badge badge--secondary">ID 101</span>

*05/12/1999 → 25/12/2005*

![Project Setup and Use](/img/manual/ProjectSetupandUse-Workflow_iDempiere_v14.0.0.png)

**Description:** Setup of projects and project reporting

**Comment/Help:** Projects allow to monitor the status and progress of sales opportunities, service projects and investment projects. A Project is a process that is undertaken which can involve multiple steps or phases and require resources from more than one area.
