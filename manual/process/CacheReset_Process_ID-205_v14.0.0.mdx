# Cache Reset

<span className="badge badge--primary">Process</span> <span className="badge badge--secondary">ID 205</span>

*16/06/2003 → 06/08/2006*

![Cache Reset](/img/manual/CacheReset-Process_iDempiere_v14.0.0.png)

**Description:** Reset Cache of the System ** Close all Windows before proceeding **

**Comment/Help:** To increase performance, iDempiere caches repeatedly used data. This process clears the local cache.
