# Quote convert

<span className="badge badge--primary">Process</span> <span className="badge badge--secondary">ID 231</span>

*04/10/2003 → 02/01/2000*

![Quote convert](/img/manual/Quoteconvert-Process_iDempiere_v14.0.0.png)

**Description:** Convert open Proposal or Quotation to Order

**Comment/Help:** You can convert a Proposal or Quotation to any other Order document type. You would use this process, if you want to maintain/keep the Proposal or Quotation. The document status needs to be In Process.

**Classname:** `org.compiere.process.CopyOrder`
