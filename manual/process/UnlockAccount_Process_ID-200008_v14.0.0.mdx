# Unlock Account

<span className="badge badge--primary">Process</span> <span className="badge badge--secondary">ID 200008</span>

*03/09/2012 → 04/07/2025*

![Unlock Account](/img/manual/UnlockAccount-Process_iDempiere_v14.0.0.png)

**Description:** Unlock Account for User

**Comment/Help:** Select a locked user from the list to unlock it.  Alternatively you can run the process without selecting any user, in this case the process will unlock all locked users from the tenant applying accordingly the policies configured for "max account lock" and "max inactive period".

**Classname:** `org.adempiere.process.ResetLockedAccount`
