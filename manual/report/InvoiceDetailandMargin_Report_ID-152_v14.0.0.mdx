# Invoice Detail and Margin

<span className="badge badge--primary">Report</span> <span className="badge badge--secondary">ID 152</span>

*27/01/2001 → 08/12/2023*

![Invoice Detail and Margin](/img/manual/InvoiceDetailandMargin-Report_iDempiere_v14.0.0.png)

**Description:** Invoice (Line) Detail and Margin Report

**Comment/Help:** The Report lists Customer and Vendor Invoice Details with Margines.
